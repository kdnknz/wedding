# Product Overview

Web-based wedding invitation application designed to be lightweight, responsive, and easy to share digitally.

## Core Purpose

- Provide a practical digital invitation that works on mobile and desktop devices
- Enable easy content updates through a centralized data file
- Allow guest interaction through a comment/greeting feature
- Deliver as a single static HTML build with no custom backend required

## Key Features

- Two-page structure: cover page and main invitation page
- Auto-play background music with play/pause controls
- Guest greeting system with comments (stored in cloud database)
- Responsive design for all screen sizes
- Shareable via link
- Hostable on static hosting services (GitHub Pages, Netlify, Vercel)

## Target Audience

Couples planning weddings who want a modern, digital invitation solution that guests can access from any device.
